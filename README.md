A Java Swing-based desktop application for efficient pet registry management using MVC architecture.

 **Features**
-**User Authentication**: Secure login using **Linear Search**. 
- **Pet Registration**: Full CRUD functionality for pet records.
- **Advanced Search**: Instant pet lookups using **Binary Search**.
- **Algorithm Analysis**: A dedicated tab to analyze sort performance.

**Implemented Algorithms**
This project demonstrates the following:
- **Selection Sort**: Orders pets by ID. 
- **Insertion Sort**: Orders pets by Name (A-Z). [
- **Merge Sort**: Orders pets by Age ($O(n \log n)$ efficiency). 
- **Binary Search**: Fast lookup for specific Pet IDs ($O(\log n)$). 
- **Linear Search**: Used for user login verification ($O(n)$).
